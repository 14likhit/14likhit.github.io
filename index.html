Projects

Forecastr

forecastr.co.uk

Forecastr is an android app designed to make it easy to view your favourite spots. Targeted at sports where the weather is key to a good session forecastr makes it quick to see which spot is best all on one screen. Forecastr includes wind, weather, swell and tide times all in one place.

Forecastr is an app I have been building as I have a keen interest in android development and wanted a personal project I could use to build my skills. I choose to build this app as I felt like its something I would use myself and also something that would set some good chalenges in lots of different areas of the build.

During the build I have utilised a number of different android libraries such as ActionBarSherlock, Dagger and Otto. As well as these the app also talks to a backend API and uses a SQLite DB.

itsgoingto.be

itsgoingto.be

itsgoingto.be is a simple voting site where a user can ask a questiong and choose some options. This question can then be shared with their friends/colleagues to vote on things like 'Where should we go for lunch?'.

The backend for this site has been build using Symfony while on the front end I have used SASS, Bourbon.io, Neat and Jquery. The front-end assets are being compiled and minified using Gulp so this can be easily auto-deployed using Heroku

I have really enjoyed working with the codebase I have created on this project as it has allowed me to build on my exisiting skills while play around and learn some new useful skills.
