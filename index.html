Projects

Forecastr

forecastr.co.uk

Forecastr is an android app designed to make it easy to view the forecast at your favourite spots. Targeted at sports where the weather is key to a good session forecastr makes it quick to see which spot is best all on one screen. Forecastr includes wind, weather, swell and tide times.

Forecastr is an app I have been building as I have a keen interest in Android development and wanted a personal project I could use to help build my skills. I chose to build this app as I felt like it's something I would use myself and also something that would set some good challenges across lots of different areas of the build.

During the build I have utilised a number of different android libraries such as ActionBarSherlock, Dagger and Otto. As well as these the app also talks to a backend API and uses a SQLite DB.

itsgoingto.be

itsgoingto.be

itsgoingto.be is a simple voting site where a user can ask a question or topic top put to a vote. This question can then be shared with their friends/colleagues to vote on things like 'Where should we go for lunch?'.

The backend for this site has been build using Symfony while on the front-end I have used SASS, Bourbon.io, Neat and JQuery. The front-end assets are being compiled and minified using Gulp so this can be easily auto-deployed on Heroku

I have really enjoyed working on this project as I have been able to create a really nice codebase that is fun to work with, and also play with some new components that I wouldn't usually have the time to work with.
